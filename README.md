# 📰 Fake News Detection System (Machine Learning)

An NLP-based text classification project that detects whether a news article is Fake or Real using TF-IDF vectorization and Logistic Regression.

#Features
- Text preprocessing and normalization
- TF-IDF feature extraction
- Logistic Regression classifier
- Achieved 98% test accuracy
- Interactive prediction interface using Jupyter widgets

#Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy
- Jupyter Notebook

#Model Pipeline
1. Data Cleaning
2. Train-Test Split
3. TF-IDF Vectorization
4. Logistic Regression Training
5. Model Evaluation
6. Interactive Prediction

#Results
- Accuracy: 98%
- Evaluated using classification metrics and confusion matrix
